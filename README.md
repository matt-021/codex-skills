# codex-skills

本仓库用于保存本地维护的 Codex skills。

## 当前技能

- `vhr-dev-guide`
- `vhr-code-review`
- `vhr-be-toolkit`
- `vhr-be-pagination`
- `vhr-fe-toolkit`
- `vhr-fe-vxe-table`
- `vhr-meta-dev`
- `vhr-project-design`
- `vhr-troubleshooting`
- `vhr-git-daily-report`

## 目录约定

- 每个 skill 使用独立目录
- 每个目录至少包含一个 `SKILL.md`

## 工具脚本

- `tools/pan-fetch.ps1`：半自动处理百度网盘项目下载

## 使用方式

将本仓库中的 skill 目录同步到 Codex 的 skills 目录后即可使用。
